🛍️ MyStore - Django E-Commerce

A fully functional, modern e-commerce web application built with Python and Django.

✨ Features

Modern UI/UX: Clean, elegant design using Google Fonts (Poppins) and custom CSS.

Product Catalog: Browse products with images, prices, and flow-card designs.

Category & Search: Dynamic filtering and search functionality.

Shopping Cart: Manage your items with a simple, dynamic cart.

User Accounts: Secure registration, login, and personalized order history.

🚀 How to Run this Project Locally

1. Clone the repository

git clone https://github.com/sarikbanyal/CodeAlpha_ProjectName.git
cd CodeAlpha_ProjectName


2. Set up the environment

python -m venv venv
source venv/Scripts/activate  # On Windows
pip install -r requirements.txt


3. Run the project

    python manage.py migrate
python manage.py createsuperuser
python manage.py runserver
